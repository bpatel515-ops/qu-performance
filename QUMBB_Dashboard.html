How to export: In your planning workbook, go to the Weekly Plan sheet → File → Save As → CSV UTF-8.

Expected columns: Date, Week, Opponent, then for each day (Sun→Sat): Volume, Session, Time. Columns 25–26 should be Weekly Total and A:C (EWMA).

Tip: The file can have up to 3 header rows — the importer will auto-detect the row containing Date and Week.
